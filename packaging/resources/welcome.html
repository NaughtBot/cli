OOBSign CLI Tools

This installer will install the OOBSign command-line tools:

  • oobsign - Main CLI for login, GPG signing, and more
  • liboobsign-sk.dylib - SSH SecurityKeyProvider for hardware-backed SSH keys

Installation Locations

  • /usr/local/bin/oobsign
  • /usr/local/lib/liboobsign-sk.dylib

After Installation

Run oobsign login to authenticate with your account.

For SSH support, add to ~/.ssh/config:

SecurityKeyProvider /usr/local/lib/liboobsign-sk.dylib
